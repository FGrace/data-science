This repository host the R code and related documentation for "Getting and Cleaning data" in Coursera.
Files in this repository:
CodeBook.md: describe the variables, the data, and any transformations or work that you performed to clean up the data
R script.R:contains code needed to perform all necessary analyses described in 5 steps.
average_alldata.txt: a tidy data set with the average of each variable for each activity and each subject
